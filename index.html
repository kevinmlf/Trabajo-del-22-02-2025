¿Qué es HTML?

HTML es el lenguaje de marcado utilizado para estructurar páginas web. Define el significado y la estructura del contenido web.📜

¿Qué es CSS?

CSS es el lenguaje de estilos que permite diseñar y dar formato a las páginas web.🎨

¿Qué es SASS?

SASS es un preprocesador de CSS que facilita la escritura de estilos con variables, mixins y más.📁
